Ce widget permet d'afficher le top 10 du box office cinéma en france sur votre blog

== Description ==
Le widget Fan de Cinéma Box Office Affiche en temps réel le classement des 10 films ayant réalisé le plus d'entrées en salle dans la semaine en France.

Il ne nécessite aucun paramétrage et s'intègre à votre thème.

Les données sont mises a jour automatiquement chaque semaine.

== Installation ==
1. Recopier le contenu de l'archive dans le répertoire des extensions (wp-content/plugins) 
2. Activer l'extension dans le backoffice de WordPress.
3. Ajoutez le widget à votre sidebar et donnez lui un titre.

C'est terminé !
